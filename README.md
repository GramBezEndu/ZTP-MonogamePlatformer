# Simple 2D platformer

### 2D platformer game made in MonoGame Framework.

### Gameplay video:

[![Platformer Game](http://img.youtube.com/vi/eRwmaZNSEi0/0.jpg)](http://www.youtube.com/watch?v=eRwmaZNSEi0 "Platformer Game")

### How To Build

1. Install [MonoGame version 3.7.1](https://community.monogame.net/t/monogame-3-7-1/11173).
2. Open project in Visual Studio 2017 or later.
3. Build the project.

<i>Note: In case of missing dependencies error, reinstall following Nuget packages: <b>MonoGame.Extended.Animations</b> and <b>MonoGame.Extended</b>.</i>
